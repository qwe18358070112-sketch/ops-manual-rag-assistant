# 政企运维知识库问答助手部署与运维手册

## 1. 运行前准备

```bash
cd /home/lenovo/projects/ops-manual-rag-assistant
uv venv
uv pip install -p .venv/bin/python -e '.[dev]'
cp .env.example .env
```

按需修改 `.env`：

- `OPS_ASSISTANT_REQUIRE_AUTH`
- `OPS_ASSISTANT_ADMIN_USERNAME`
- `OPS_ASSISTANT_ADMIN_PASSWORD`
- `OPS_ASSISTANT_ADMIN_TOKEN`
- 需要大模型重写时，再补 `OPS_ASSISTANT_LLM_*`

## 2. 本地启动

```bash
./scripts/start_local.sh
```

默认地址：

- `http://127.0.0.1:8000/app`

## 3. 资料库运维命令

### 初始化资料库

```bash
.venv/bin/python scripts/manage_library.py init
```

### 全量重建

```bash
.venv/bin/python scripts/manage_library.py rebuild
```

### 单文档重建

```bash
.venv/bin/python scripts/manage_library.py reindex <doc_id>
```

### 导出资料列表

```bash
.venv/bin/python scripts/manage_library.py export --kind documents --format csv
.venv/bin/python scripts/manage_library.py export --kind documents --format json
```

### 导出分析结果

```bash
.venv/bin/python scripts/manage_library.py export --kind analysis --format json
```

## 4. 目录说明

- 上传文件：`data/raw/uploads/`
- 解析结果：`data/extracted/`
- 切块结果：`data/chunks/`
- 检索索引：`data/indexes/`
- 资料库元数据：`data/library/metadata.sqlite3`
- 导出结果：`data/exports/`
- 访问日志：`data/logs/access.log`

## 5. 常见运维动作

### 上传后没有检索到新资料

1. 检查资料库页面中文档状态是否为 `indexed`
2. 若状态不是 `indexed`，查看 `last_error`
3. 对该文档执行“重新索引”
4. 必要时执行全量重建

### 全量重建后结果不刷新

- 当前系统会在重建后自动清空运行时缓存
- 若浏览器仍显示旧结果，刷新页面即可

### 权限控制

启用管理员控制：

```bash
OPS_ASSISTANT_REQUIRE_AUTH=true
OPS_ASSISTANT_ADMIN_USERNAME=admin
OPS_ASSISTANT_ADMIN_PASSWORD=change-me
```

禁用管理员控制：

```bash
OPS_ASSISTANT_REQUIRE_AUTH=false
```

## 6. 本机验收建议

```bash
.venv/bin/python -m unittest discover -s tests -p 'test_*.py' -v
.venv/bin/python scripts/run_acceptance_checks.py
```

再在浏览器打开：

- `/app`
- `/app/search`
- `/app/library`
- `/app/timeline`
- `/app/topics`

重点检查：

- 上传是否成功
- 文档是否进入资料库
- 检索是否能命中新资料
- 导出是否成功
- 日志文件是否写入
