## 2. 可直接替换进简历的项目文本

### 政企运维知识库问答助手

项目角色：独立设计与开发  
技术方向：Python + FastAPI + SQLite FTS + TF-IDF + RAG + 文档解析

- 基于综治中心操作手册、应急预案、周报和月报，搭建面向政企驻场运维场景的知识库问答系统，支持 `docx/pdf/xlsx/txt` 文档导入、结构化切块、混合检索和引用式答案生成。
- 设计“手册问答 / 历史案例检索”双模式，支持 `system_name/source_type/tag/date` 多维过滤，提升运维资料定位效率与历史故障复盘效率。
- 实现 section 级标签、案例时间轴、专题详情、案例详情和案例到问答联动，支持从答案引用回到原始案例记录，增强结果可解释性。
- 项目当前已完成本地可运行版本和接口回归测试，可用于展示真实运维场景下的 AI 应用落地能力。
