## 本轮代码审查发现的问题

### 1. 测试链路在当前环境不可直接运行

问题：

- `tests/test_api_regression.py` 顶层直接导入 `TestClient`
- 当前 `.venv` 未装 `httpx` 时，测试会在导入阶段直接报错，而不是给出清晰提示

修复：

- 测试文件改为在缺失 `httpx` 时跳过并给出明确原因
- 当前环境已补装 `httpx`

### 2. 检索对不相关问题存在误召回

问题：

- 如 `不存在的词` 这类问题，原逻辑仍会返回手册内容
- 原因是中文 token 中存在低信号 bigram，导致局部匹配被误认为相关

修复：

- 在混合检索后增加相关性门槛
- 对包含功能词的低信号 query token 做过滤
- 增加“无关问题返回 0 结果”的回归测试

### 3. 手册类引用缺少反向跳转详情页

问题：

- 原来案例类引用可以跳到 `case-detail-page`
- 手册类引用只能看到来源名，无法回到手册原文章节

修复：

- 新增 `manual_views.py`
- 新增 `/manual-detail` 和 `/manual-detail-page`
- 引用链路增加手册章节详情跳转
